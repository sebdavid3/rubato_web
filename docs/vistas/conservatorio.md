## Notas de implementación

- La sección de Trinity College London es un diferenciador clave (único en Colombia); destacar visualmente con `card` y posiblemente un badge/logotipo de Trinity.
- El pénsum académico usa un formato de tabs o accordion para no abrumar con 3 módulos desplegados.
- Los datos de fechas y costos cambian cada semestre. Deben ser editables (CMS o archivo de datos), no hardcodeados.
- Los instrumentos pueden mostrarse como grid de cards con icono o miniatura.
- "Mis Primeras Notas" es un programa especial para niños de 4-6 años; considerar una card con diseño diferenciado.
